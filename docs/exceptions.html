exception gemma.SuppressedErrors(*args)[source]¶

Errors occurred while Cartographer mapped data

Attributes:

  • errors ( List[BaseException] ): list of errors which occurred.

  • chart_partial ( List[Tuple["Course", Any]] ): If raised from Surveyor.chart() raising a NonNavigableError for some elements, a partial chart of the successfully traversed data will be stored here.
